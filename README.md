Cerinta este:
Creaţi un program care simulează modul de printare a documentelor multiple de către un printer. Un document este carcaterizat prin titlu şi conţinut. Programul are următoarele funcţionalităţi: 
a. Adăugarea unui nou document spre printare. 
b. Stergerea unui nou document spre printare. 
c. Schimbarea priorităţii unui document la printare – al k-lea document devine următorul ce urmează a fi procesat cu prioritate de catre printer. 
d. Funcţia Procesare() care procesează lista de documente de printat în ordinea adăugării lor. Considerăm operaţia de printare ca fiind afişarea pe ecran pentru fiecare document în formatul: “titlu : continut”. 

E doar o coada si se poate rezolva, in main e pus in comentarii ca este si un program cu stiva prezentat.
